# Salary-Prediction
Salary prediction is a real-world application of Machine Learning that estimates a person’s salary based on factors like experience, education, skills, and job role. It is a regression problem because salary is a continuous value.

A model is trained using past data, where preprocessing is done before applying algorithms like Linear Regression. The trained model can then predict salary for new inputs.

Its performance is measured using metrics like MAE, MSE, RMSE, and R². It is widely used in HR and job platforms, but its accuracy depends on data quality.

In short, it is an important application of Data Science for making salary-related decisions.
